Auto-expand image widgets in Dashboard Widgets page, making images visible.

== Description ==

This plugin helps you manage image widgets easier. If you manage many image widgets, this plugin will help you see them quicker without having to expand each one. It's useful if you like to manually edit your image widgets often, and also if you don't have titles on your image widgets.

== Installation ==
<ol>
<li>Install easily with the WordPress plugin control panel or manually download the plugin and upload to the `/wp-content/plugins/` directory.</li>
<li>Activate the plugin through the 'Plugins' menu in WordPress.</li>
</ol>
